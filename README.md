# Warehouse Layout Designer

물류 센터 레이아웃을 시각적으로 설계할 수 있는 웹 기반 디자인 도구입니다.

## 주요 기능

- 🎨 **드래그 앤 드롭** 방식의 직관적인 설비 배치
- 🔗 **노드 연결**을 통한 물류 흐름 설계
- 📦 **다양한 설비 유형** 지원 (랙, 컨베이어, AGV 로봇, 작업 스테이션, 게이트)
- 🗺️ **미니맵**으로 전체 레이아웃 한눈에 파악
- ⚡ **실시간 편집** 및 미리보기
- 🎯 **상태 관리** (active, idle, error)

## 기술 스택

- **React** 18.3.1 - UI 라이브러리
- **TypeScript** 5.6.2 - 타입 안정성
- **ReactFlow** 11.11.4 - 노드 기반 다이어그램
- **Vite** 6.0.5 - 빌드 도구
- **Tailwind CSS** 3.4.17 - 스타일링
- **dak-krds** 0.1.11 - 커스텀 컴포넌트 라이브러리
- **pnpm** - 패키지 관리

## 사용 방법

1. **설비 선택**: 상단 드롭다운에서 추가할 설비 유형 선택
2. **설비 추가**: "+ 설비 추가" 버튼 클릭
3. **배치**: 생성된 노드를 드래그하여 원하는 위치에 배치
4. **연결**: 노드의 연결점을 다른 노드로 드래그하여 물류 흐름 설계

## 설비 유형

| 아이콘 | 유형 | 설명 |
|------|------|------|
| 📦 | 랙 (Rack) | 상품 보관 설비 |
| ➡️ | 컨베이어 (Conveyor) | 이동 설비 |
| 🤖 | AGV 로봇 (Robot) | 자동 운반 설비 |
| 🏭 | 작업 스테이션 (Workstation) | 작업 공간 |
| 🚪 | 게이트 (Gate) | 입출고 지점 |

## 상태 표시

- **Active** (활성) - 현재 가동 중인 설비
- **Idle** (유휴) - 대기 상태의 설비
- **Error** (오류) - 문제가 발생한 설비
